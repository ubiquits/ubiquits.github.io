0.3.37 (2016-06-17)

Features

  • docs: Added contribution, developer and license guidelines, updated readme (f04b573)
  • routes: Renamed @Action decorator to @Route (8054c10)

BREAKING CHANGES

  • routes: All imports and instances of @Action need to be renamed to @Route

Example

import { Action } from 'ubiquits/core/server';
[...]
@Action('GET', '/test')
public method()

becomes

import { Route } from 'ubiquits/core/server';
[...]
@Route('GET', '/test')
public method()
